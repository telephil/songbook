# songbook

Explorer for guitar tabs in text format

# Overview

A web UI that displays all tabs in a given directory

# Usage

To start the web application use:
```lisp
(ql:quickload :songbook)
(songbook:browse #p"path/to/tabs")
```

The default port is 40000
